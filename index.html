Pygame project

My implementation of a simple turnbased RPG game using python and pygame.

Uses WASD controls, with the green square being the player character.

Makes use of OOP to create Players, Enemies and the Map.

Map is represented as a 2d array, with each inner list representing a row, and each element representing a single square in the map.

Map is randomly generated with rooms and corridors, accepting a width, height and number of rooms as parameters.

Battle system is implemented in the console, beginning when the player and enemy enter adjacent squares. The player can attack or heal until the player or enemy lose.

If the player wins, a new map and enemy is generated. Otherwise, the game quits.

See it here on replit.
